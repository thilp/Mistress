NAME

Mistress::Obj::DistSet - Set of Mistress::Obj::Dist objects.

VERSION

version 0.001

SYNOPSIS

DESCRIPTION

This object builds (and then represents) a set of Mistress::Obj::Dist objects.

The set is indirectly build through plugins that implement the Mistress::Role::DistGatherer role. Each of these plugins produces a list of Dist objects and all these lists are merged here.

Thanks to a blacklist system, only "fresh" distributions (i.e. that have not been in a DistSet without being "smoked" later) are added to this set.
